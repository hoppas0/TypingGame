  A simple Typing game. <br>
  `It's only for personal learning and noncommercial use. If this game infringes the copyright, please let me know.`

# Requirement
* Python 3.9
* Pygame 2.5

# How To Start Game
$ python main.py

# How to Play
* Earn points for spelling words
